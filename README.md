# A2A Samples

Sample applications demonstrating how to integrate with an A2A-compatible endpoint using the [A2A (Agent-to-Agent) protocol](https://google.github.io/A2A/).

## Samples

| Sample | Language | Description |
|--------|----------|-------------|
| [rust-cli](rust-cli/) | Rust | Interactive CLI for A2A sessions |
| [A2A Chat](A2A%20Chat/) | Swift | iOS/macOS chat app for A2A sessions |

## Prerequisites

- A Microsoft 365 account
- An Azure AD app registration configured for device code flow (see sample-specific instructions)
