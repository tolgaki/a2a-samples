# Work IQ Samples

Sample applications demonstrating how to integrate with [Microsoft Work IQ](https://learn.microsoft.com/microsoft-365-copilot/extensibility/work-iq-overview) using the [A2A (Agent-to-Agent) protocol](https://google.github.io/A2A/).

## Samples

| Sample | Language | Description |
|--------|----------|-------------|
| [rust-cli](rust-cli/) | Rust | Interactive CLI for A2A sessions with Work IQ |
| [A2A Chat](A2A%20Chat/) | Swift | iOS/macOS chat app for A2A sessions with Work IQ |

## Prerequisites

- A Microsoft 365 account with access to Work IQ
- An Azure AD app registration configured for device code flow (see sample-specific instructions)
